C#

¿Qué es C#?

Es un lenguaje de programación moderno desarrollado por Microsoft.

¿Para qué sirve?

Sirve para crear aplicaciones de escritorio, videojuegos (con Unity) y aplicaciones web (.NET).

Función

Su función principal es facilitar el desarrollo rápido, robusto y seguro de software dentro del ecosistema Windows y .NET, combinando simplicidad con potencia para crear desde pequeñas herramientas hasta grandes sistemas empresariales.
